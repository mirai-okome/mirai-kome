# mirai.F - 兵庫県丹波市産 特別栽培米 公式ホームページ

## プロジェクト概要

兵庫県丹波市産の特別栽培米を販売する「mirai.F」の公式ホームページです。
農薬を大幅に削減し、自然栽培に近づけた安心・安全で美味しいお米をお届けします。

## 🌾 プロジェクトの特徴

- **1ページ完結型のレスポンシブデザイン**
- **ブランドカラーを統一したクリーンなUI**
- **公式LINE導線を3箇所に設置**
- **商品情報・価格・送料を明確に表示**
